# ЛАБОРАТОРНАЯ РАБОТА №4

**Тема:** RAII и управление ресурсами

---

## 1. Задание

Реализовать библиотеку `resource_core`, демонстрирующую корректную работу с внешними ресурсами на примере файлов.

---

## 2. Функциональные требования

### 2.1. Пространства имён

Вся библиотека размещается в пространстве имён `lab4::resource`.

### 2.2. Библиотека resource_core

Минимальные сущности:
- `ResourceError` — собственный тип исключений для ошибок, возникающих при работе с ресурсами.
- `FileHandle` — класс, который инкапсулирует внешний ресурс и управляет его временем жизни по принципу RAII.
- `ResourceManager` — класс, который предоставляет доступ к ресурсам и хранит их в кеше.

Ожидается, что:
- ресурс захватывается при создании объекта или при выполнении специальной операции открытия;
- ресурс автоматически освобождается при уничтожении объекта;
- объект корректно ведёт себя при передаче владения;
- ошибки работы с ресурсом обрабатываются через `ResourceError`.

### 2.3. Тесты

Добавьте юнит-тесты, проверяющие корректное владение, разделяемое использование ресурсов и работу кеша.

---
